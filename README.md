# MYSQL--Day2--intermediate-JOINS-and-STRING-FUNCTIONS-
This project demonstrates intermediate MySQL skills including Joins (INNER, LEFT, RIGHT, SELF JOIN) and String Functions. It uses a structured student dataset to perform data analysis, transformations, and relational queries.


##  Overview
This project demonstrates intermediate-level MySQL concepts using a student dataset.

##  Topics Covered
- Joins (INNER JOIN, LEFT JOIN, RIGHT JOIN, SELF JOIN)
- String Functions (UPPER, LOWER, LENGTH, CONCAT, SUBSTRING, TRIM, REPLACE, INSTR, LEFT, RIG

## Key Learnings
- Understanding relationships between tables using joins  
- Performing data transformation using string functions  
- Writing optimized and structured SQL queries
